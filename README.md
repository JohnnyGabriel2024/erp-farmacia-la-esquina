# ERP Farmacia La Esquina

Sistema ERP desarrollado en Python Flask y SQLite para la gestión de una farmacia de barrio dedicada al comercio minorista de productos farmacéuticos, medicamentos, artículos de perfumería, cosmética y cuidado personal.

## Módulos del sistema

- Dashboard Ejecutivo
- Categorías
- Productos
- Proveedores
- Compras
- Ventas
- Inventario
- Movimientos
- Analítica

## Tecnologías

Python  
Flask  
SQLite  
HTML  
CSS  
Jinja Templates

## Instalación

Instalar dependencias:

pip install flask

## Ejecución

Ejecutar el sistema:

python app.py

Abrir en el navegador:

http://127.0.0.1:5000

## Base de datos

La base de datos SQLite se crea automáticamente al ejecutar el sistema.

Archivo:

database.db

## Descripción del negocio

Farmacia La Esquina es un establecimiento farmacéutico comunitario orientado a la atención sanitaria primaria y a la venta minorista de medicamentos, productos de cuidado personal, cosmética y perfumería.

Este ERP permite administrar inventario, compras a proveedores, ventas, movimientos de stock y analítica comercial.
